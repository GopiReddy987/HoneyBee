# HoneyBee

HoneyBee is a simple Angular demo that showcases basic routing and a minimal
shopping cart. The navigation includes links to **Home**, **About**,
**Products**, **Cart**, and **Contact** pages. The product list features three
static honey jar sizes (¼&nbsp;kg, ½&nbsp;kg, and 1&nbsp;kg) with an alert-based
"Add to Cart" button.

The cart view summarizes item totals and provides a "Proceed to Payment" button
as a placeholder for future Stripe integration.

See [docs/cart-basics.md](docs/cart-basics.md) for a checklist of recommended
cart features.
