# Basic Shopping Cart Features

A minimal e‑commerce cart should include:

- **Add to Cart** – Allow customers to add products from the catalog.
- **Remove Items** – Users can remove individual products from the cart.
- **Quantity Updates** – Provide controls to change the quantity of items.
- **Cart Summary** – Display subtotals and total price for all items.
- **Persist Cart** – Keep the cart contents while browsing different pages.
- **Checkout Link** – A clear path for customers to proceed to checkout.

These features create a usable foundation that can be extended with promo codes, shipping options, and other enhancements.
